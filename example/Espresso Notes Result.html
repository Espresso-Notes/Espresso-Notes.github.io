Compiler Data Classes

These were the DataClasses utilized by the Compiler. This serves as an example of the code syntax highlighting for a different language.

@dataclass
class NotesDocument:
    documentID : str
    title : str
    author : str
    lastModified : str
    content : list

@dataclass
class ContentBlock:
    contentID : int
    index : int
    blockType : str

@dataclass
class MarkdownBlock(ContentBlock):
    text : str

@dataclass
class CodeBlock(ContentBlock):
    code : str
    language : str

@dataclass
class LaTeXBlock(ContentBlock):
    text : str
